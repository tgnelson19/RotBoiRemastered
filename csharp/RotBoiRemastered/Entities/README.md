Runtime game objects. Planned mapping from the Python source:

- `Player.cs` <- `character.py` (player-related pieces) + `characterStats.py`
- `Bullet.cs` <- `bullet.py`
- `Enemy.cs` + subtypes <- `enemyTypes.py`, `enemy.py`
- `EnemyProjectile.cs` <- `enemyProjectile.py`
- `ExperienceBubble.cs` <- `experienceBubble.py`
- `LootCrate.cs` <- `lootCrate.py`
- `DamageText.cs` <- `damageText.py`
- `ProjectilePortal.cs` <- `projectilePortal.py`
- Boss types <- `bossTypes.py`
